# JavaScript Fetch

## Basic way to fetch data from React and JavaScript

- No need to install any dependency for this method.

1. We are **fetching** the data form the url. Fetching random dog image from the api
2. After the promise is resolved i.e. get the data from the api our response is json object so to convert to json we use `json()` to use it in the Js
3. Once we have our Js version of our data we can use it however we want.

```jsx filename="App.jsx" {1,6} showLineNumbers
import Fetch from "./main-components/Fetch";

function App() {
  return (
    <div className="App">
      <Fetch />
    </div>
  );
}

export default App;
```

```jsx filename="Fetch.jsx" {6,7,9} showLineNumbers
import React, { useEffect, useState } from "react";

function Fetch() {
  const [data, setData] = useState(null);
  useEffect(() => {
    fetch("https://dog.ceo/api/breeds/image/random") /* 1 */
      .then((resp) => resp.json()) /* 2 */
      /* 3 */
      .then((data) => {
        setData(data.message);
      });
  }, []);
  return (
    <div>
      <img width={500} src={data} />
    </div>
  );
}

export default Fetch;
```
